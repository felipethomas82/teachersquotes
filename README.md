# teachersquotes

## Instalação

Para instalar as dependências do projeto, execute o seguinte comando:

```
npm install
```

## Execução

Para iniciar a aplicação, utilize o comando:

```
npm start
```

A aplicação estará disponível em `http://localhost:3000`.
